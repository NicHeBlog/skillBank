# 衣图转换大师 (Garment Vision Pro)

## 核心任务

将服装设计手绘稿/草图转化为5项专业输出，最终整合为一份完整的PDF文档：
1. 高清平铺样衣图 (Flat Lay)
2. 工艺与色彩参考 (Tech & Color)
3. 国内市场版 (Domestic Lookbook)
4. 海外市场版 (Global Lookbook)
5. 整合PDF文档 (Garment Portfolio PDF)

## 输入处理

### 识别内容
- 轮廓：领口、袖子、长短、褶皱
- 款式结构：版型、剪裁特征

### 信息收集
| 场景 | 操作 |
|------|------|
| 只上传图片 | 询问面料类型、颜色偏好、适用性别 |
| 图片+完整信息 | 直接启动五项输出 |
| 未上传图片 | 提示需要手绘稿/草图 |

## 输出规范

### 阶段一：基础图生成

#### 1. 高清平铺样衣图
- 背景：纯白/浅灰色，光影均匀
- 材质表现：参见 [references/fabric-guide.md](references/fabric-guide.md)
- 要求：正面视角，完整展示服装细节

#### 2. 工艺与色彩参考图
- 2-3个核心Pantone色号标注
- 面料物理特性可视化
- 关键工艺细节特写
- 图片中不要出现中文

### 阶段二：图生图一致性保持

**核心原则**：使用阶段一生成的"高清平铺样衣图"作为图生图的参考图像，确保国内市场版和海外市场版的服装款式、颜色、面料质感保持高度一致。

**图生图基础参数**：
```
--image-url: [阶段一生成的高清平铺样衣图URL]
--image-mode: IMAGE_REFERENCE
--strength: [根据服装类型动态选择，见下表]
```

**分层strength参数策略**（根据服装结构复杂度）：

| 服装类型 | 结构复杂度 | 推荐strength | 说明 |
|----------|------------|--------------|------|
| 基础款 (T恤/衬衫/简单连衣裙) | 低 | **0.82** | 较高值允许更多模特/场景变化 |
| 结构款 (西装/风衣/大衣) | 中 | **0.80** | 平衡结构保持与场景适配 |
| 复杂款 (礼服/多层裙) | 高 | **0.77** | 中等约束保持复杂结构 |
| 特殊面料款 (刺绣/珠片/蕾丝) | 极高 | **0.75** | 强约束保持精细纹理细节 |

**面料类型参数微调**：

| 面料 | 光泽度 | strength微调 | 原因 |
|------|--------|--------------|------|
| 丝绸 | 高 | -0.02 | 光泽反射需更精确控制 |
| 棉麻 | 低 | +0.02 | 纹理允许更灵活变化 |
| 针织 | 低 | 基准 | 标准参数即可 |
| 牛仔 | 低 | 基准 | 标准参数即可 |
| 皮革 | 中高 | -0.01 | 光泽与纹理需平衡 |
| 呢绒 | 低 | 基准 | 标准参数即可 |

**图生图Prompt一致性约束词**（必须添加到生成提示中）：
- `"exactly the same garment"` / "完全相同的服装"
- `"identical clothing design"` / "完全一致的设计"
- `"maintain all details and textures"` / "保持所有细节和纹理"
- `"preserve exact silhouette"` / "保持精确的轮廓"

**一致性控制要点**：
- 服装轮廓线：必须与原图100%一致
- 面料纹理：保持相同的材质表现
- 颜色准确度：Pantone色号严格一致
- 细节特征：纽扣、拉链、缝线位置不变

**已知限制与缓解措施**：

| 限制类型 | 现象 | 缓解措施 |
|----------|------|----------|
| 复杂褶皱 | 平铺与上身状态褶皱形态不同 | 正常现象，在PDF中说明 |
| 光泽面料 | 不同光照下反射有差异 | 统一光源描述，降低strength |
| 精细装饰 | 刺绣/珠片可能轻微偏移 | 特殊面料款使用最低strength值 |

**最低可接受一致性阈值**：85%

#### 3. 国内市场版 (基于图生图)
- **参考图像**：高清平铺样衣图
- 模特：亚洲面孔，淡妆自然
- 场景：简约影棚/武康路街拍/现代室内
- 氛围：气质、精致、清新
- 保持服装与参考图完全一致

#### 4. 海外市场版 (基于图生图)
- **参考图像**：高清平铺样衣图
- 模特：欧美/多族裔，健康肤色
- 场景：巴黎/纽约街拍/都市风格
- 氛围：力量感、时尚张力、多元化
- 保持服装与参考图完全一致

### 阶段三：PDF文档生成

#### 5. 整合PDF文档 (Garment Portfolio)

使用 `aily-pdf` 技能生成专业PDF文档，结构如下：

**封面页**
- 产品名称（从用户输入提取）
- 设计日期
- 版本号

**目录页**
- 文档结构导航

**第一章：高清平铺样衣图**
- 图片展示
- 设计概述
- 设计亮点

**第二章：工艺与色彩参考**
- 图片展示
- Pantone色号列表（含色块预览）
- 面料技术参数表
- 工艺说明

**第三章：国内市场版**
- 图片展示
- 场景说明（场景氛围、模特风格、品牌调性）
- 目标人群分析
- 营销建议（渠道策略、内容营销）

**第四章：海外市场版**
- 图片展示
- 场景说明（场景氛围、模特风格、品牌调性）
- 目标人群分析
- 营销建议

**附录A：面料渲染指南摘要**

**附录B：社交媒体标签建议汇总**

## 执行流程

1. **输入确认** → 收集面料、颜色、性别信息
2. **生成平铺图** → 文生图创建高清样衣图（保存URL）
3. **生成工艺图** → 图生图（参考平铺图）创建工艺与色彩参考图
4. **生成国内版** → 图生图（参考平铺图）创建国内市场版
5. **生成海外版** → 图生图（参考平铺图）创建海外市场版
6. **生成PDF** → 调用 `aily-pdf` 整合所有图片和描述
7. **交付成果** → 展示PDF并提供下载链接

## 语言规范
- 默认中文分析
- 保留专业术语英文：Pantone、Flat Lay、Lookbook
- 面料描述使用行业标准词汇

## 输出格式

### 图片输出（5张）
| 序号 | 图片类型 | 生成方式 | 一致性基准 |
|------|----------|----------|------------|
| 1 | 高清平铺样衣图 | 文生图 | - |
| 2 | 工艺与色彩参考 | 图生图 | 参考图片1 |
| 3 | 国内市场版 | 图生图 | 参考图片1 |
| 4 | 海外市场版 | 图生图 | 参考图片1 |

### PDF文档输出
- 格式：A4尺寸，专业排版
- 包含：全部4张高清图片 + 详细描述内容
- 用途：可打印、可分享的专业服装档案
